## Tab Group bookmark

Save your chrome tab groups into bookmarks and
- reopen from other laptop with access to shared bookmarks
- reopen later when you need


#### Npm scripts: 
- build
- dev

#### Use extension:
1. run `npm build`
2. archive out folder
3. import into Chrome
